## Features

1. Localization
2. Provides best solutions to the farmer's problem using NLP algorithm and knowledge base repository.
3. Crop Recommendation
4. Fertilizer Recommendation
5. Disease Prediction
6. Weather Forcasting
7. Voice Assistance
